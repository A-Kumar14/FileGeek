# FileGeek Brand Guidelines

## Brand Name
**FileGeek**

Always written as a single word with "F" and "G" capitalized. Reflects a high-performance, developer-centric tool for document intelligence.

### Acceptable Usage
- FileGeek
- FILEGEEK 

### Unacceptable Usage
- File Geek (with space)
- filegeek (all lowercase, except in terminal-style UI)
- Filegeek (lowercase 'g')

---

## Logo Usage

### Clear Space
Maintain clear space around the logo equal to the height of the "F" in the logo on all sides.

### Minimum Size
- Digital: 120px width minimum
- Print: 1 inch / 25mm width minimum

### Logo Don'ts
- Do not stretch or distort the logo.
- Do not add effects like shadows, gradients, or outlines; keep it flat and brutalist.
- Do not use rounded or "friendly" fonts; stick to technical monospaced type.
- Do not place the logo on busy or non-neutral backgrounds.

---

## Color Palette

### Primary Color
**Geek Green (Accent)**
- HEX: #00FF00
- RGB: 0, 255, 0
- Use as: Primary brand accent, active tab indicators, primary buttons, and successful status alerts.

### Secondary Colors
**Carbon (Base)**
- HEX: #0D0D0D
- RGB: 13, 13, 13
- Use as: Secondary backgrounds, paper components, and card containers.

**Brutalist Black**
- HEX: #000000
- RGB: 0, 0, 0
- Use as: Primary background, TopBar background, and main layout container.

### Neutral Colors
**Terminal White**
- HEX: #E5E5E5
- Use as: Primary text, active icons, and high-emphasis labels.

**Steel Gray (Border)**
- HEX: #333333
- Use as: All borders, dividers, and inactive UI elements.

**Dim Gray**
- HEX: #888888
- Use as: Secondary text, captions, and disabled states.

---

## Typography

### Primary Font
**JetBrains Mono** (Fallbacks: Fira Code, Courier New, monospace)

### Font Weights
- **Bold (700)**: Headings, button labels, and command prefixes.
- **Medium (500)**: Subheaders and list items.
- **Regular (400)**: Body text and AI reasoning logs.

### Font Sizes
- H4: 32px / 1.5rem (700 weight)
- Body 1: 16px / 1rem (1.6 line height)
- Body 2 (UI): 14px / 0.875rem
- Small (Mono): 12px / 0.75rem
- Caption: 10px / 0.65rem

---

## UI Elements

### Buttons
**Brutalist Button**
- Background: Black (#000000)
- Border: 1px solid #333333
- Text: Geek Green (#00FF00)
- Border Radius: 0 (Zero ornamentation)
- Hover: Background #333333 or Border #00FF00

### Cards & Panels
- Background: Carbon (#0D0D0D)
- Border: 1px solid #333333
- Border Radius: 0
- Shadow: None (Flat design)

### Input Fields (Terminal Style)
- Background: Transparent
- Border: 1px solid #333333
- Focus Border: Geek Green (#00FF00)
- Prefix: `//` or `>`

---

## Iconography
- **Style**: Lucide React or Outlined system icons.
- **Weight**: Thin to medium stroke width.
- **Color**: Geek Green for active/accent, Dim Gray for inactive.

---

## Tone of Voice
- **Technical**: Precise, high-context, and efficient.
- **Agentic**: Proactive and transparent about AI reasoning processes.
- **Minimalist**: Direct, no fluff, and zero marketing jargon.
